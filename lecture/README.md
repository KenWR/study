## 강의

(SNUON) 운영체제의 기초: 쉽게 배우는 운영체제 원리    

이 강의는 기본적으로 자료구조, 알고리즘, 프로그램 언어, 컴파일러, 컴퓨터 구조, 하드웨어를 선행지식으로 기반하여 진행되는 강좌이다
